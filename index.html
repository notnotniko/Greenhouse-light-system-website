Introduction

This project uses an Arduino Uno, a GL5528 Photoresistor (LDR), and white LEDs to provide extra light to plants in dark environments.

Circuit Diagram

The photoresistor measures light levels; the Arduino activates LEDs to provide necessary light.

Environmental Impact Assessment

  • Materials: The Arduino Uno is RoHS and WEEE compliant, limiting hazardous substances.
  • Energy Efficiency: The circuit draws ~200 mA, so plugging in through USB is fine, however a transistor is required to not harm the Arduino.
  • Reusability: All components are modular and reusable, reducing electronic waste.
  • Packaging & Shipping: Ordering all components from a single supplier helps minimize carbon footprint.

Arduino Code

The following Arduino code controls the system, lighting LEDs based on the readings from the photoresistor.

  
  
//Lighting system with Photoresistor (LDR)
//Author: Patrick Kiernicki
//Nov 18, 2025
//If the LDR detects the light level to be under a certain value (400), it will turn the LEDs on.

//LDR pin
int sensorPin = A0;
//LEDs pin
int ledControlPin = 9;

void setup() {
  pinMode(ledControlPin, OUTPUT);
}
void loop() {
  int lightLevel = analogRead(sensorPin); //measures light level
  if (lightLevel < 20) {
    digitalWrite(ledControlPin, HIGH); //turns LEDs on
  } else {
    digitalWrite(ledControlPin, LOW); //turns LEDs off
  }
}

Conclusion

The system effectively demonstrates turning on multiple LEDs in low light conditions using Arduino. It's affordable, educational, and sustainable through reusable components.
